## 📝 Giới thiệu dự án (Project Description)
**FixNow** là ứng dụng di động hỗ trợ đặt lịch sửa chữa các thiết bị điện tử (điện thoại, laptop, đồ gia dụng). Ứng dụng đóng vai trò cầu nối giúp người dùng tìm kiếm kỹ thuật viên uy tín một cách nhanh chóng và minh bạch.

---

## 👥 Đối tượng người dùng (Target Users)
* **Khách hàng (Customers):** Tìm kiếm dịch vụ, tạo yêu cầu và theo dõi sửa chữa.
* **Kỹ thuật viên (Technicians):** Tiếp nhận yêu cầu, quản lý công việc và cập nhật tiến độ.
* **Quản trị viên (Admin):** Kiểm soát toàn bộ hệ thống, người dùng và các giao dịch.

---

## ✨ Tính năng chính (Main Features)
| Tính năng | Mô tả |
| :--- | :--- |
| 🔐 **Authentication** | Đăng ký, đăng nhập và bảo mật tài khoản người dùng. |
| 🛠️ **Booking** | Khách hàng tạo yêu cầu sửa chữa kèm mô tả lỗi và hình ảnh. |
| ✅ **Job Acceptance** | Kỹ thuật viên xem danh sách đơn hàng và chấp nhận việc làm. |
| 📍 **Tracking** | Theo dõi trạng thái sửa chữa theo thời gian thực. |
| ⭐ **Review & Rating** | Đánh giá chất lượng dịch vụ của kỹ thuật viên sau khi hoàn thành. |

## 🛠 Công nghệ sử dụng (Technology Stack)

### **Frontend**
* **Framework:** [React Native](https://reactnative.dev/) (Expo)
* **UI Components:** React Native Paper / Elements

### **Backend**
* **Framework:** [Spring Boot](https://spring.io/projects/spring-boot) (Java)
* **Security:** Spring Security / JWT

### **Database**
* **Hệ quản trị:** [MySQL](https://www.mysql.com/)
* **ORM:** Spring Data JPA

## 🚀 Hướng dẫn cài đặt (Installation)
*(Phần này bạn có thể bổ sung sau khi hoàn thiện code)*
1. Clone repository: `git clone https://github.com/your-username/FixNow.git`
2. Cài đặt Backend: Chạy file `mvn install` trong thư mục backend.
3. Cài đặt Frontend: Chạy `npm install` và `npx expo start`.
