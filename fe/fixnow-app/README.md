
# FixNow – Moble Repair Booking App

## 1. Project Description
FixNow is a mobile application that allows users to book repair services 
for electronic devices such as phones, laptops, and home appliances.

## 2. Target Users
- Customers
- Technicians
- Admin

## 3. Main Features
- User authentication (login/register)
- Create repair requests
- Technician accepts repair jobs
- Track repair status
- Review technicians

## 4. Technology Stack
- Frontend: React Native (Expo)
- Backend: Spring Boot (Java)
- Database: MySQL
